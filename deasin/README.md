# 대신네트웍스 순천지사 홈페이지

대신네트웍스(주) 순천지사의 공식 홈페이지입니다. IT 인프라, 네트워크, 보안 솔루션 등 전문적인 IT 서비스를 제공합니다.

## 📋 프로젝트 개요

- **프로젝트명**: 대신네트웍스 순천지사 홈페이지
- **목적**: 순천 지역 고객을 위한 IT 서비스 홍보 및 온라인 견적 문의 채널 제공
- **개발 기간**: 2024년 2월
- **버전**: 1.0.0

## ✨ 주요 기능

### 현재 구현된 기능

1. **메인 페이지**
   - 히어로 섹션 with 그라데이션 배경
   - 스크롤 애니메이션 효과
   - 반응형 네비게이션 메뉴

2. **회사소개 섹션**
   - 순천지사 소개 및 특징
   - 실시간 카운팅 애니메이션 통계
   - 거래 기업, 완료 프로젝트, 전문 인력, 경력 통계

3. **서비스 안내**
   - 6가지 주요 서비스 카드
   - 네트워크 구축, 정보보안, 서버/스토리지, 클라우드, 영상보안, 유지보수
   - 각 서비스별 상세 설명 및 제공 항목

4. **주요 고객사**
   - 카테고리별 고객사 분류 (공공기관, 제조/산업, 의료/교육)
   - 호버 효과가 적용된 고객사 카드

5. **견적 요청 시스템** ⭐
   - 온라인 견적 요청 폼
   - 실시간 데이터베이스 저장
   - 이메일 유효성 검사
   - 전화번호 자동 포맷팅
   - 견적 문의 프로세스 4단계 안내

6. **공지사항** ⭐
   - 동적 공지사항 로딩
   - NEW 뱃지 자동 표시 (7일 이내)
   - 상단 고정 기능 (is_pinned)
   - 샘플 공지사항 4건 등록

7. **오시는 길**
   - 순천지사 연락처 및 주소
   - 업무시간 안내
   - 지도 영역 (API 연동 준비됨)

8. **반응형 디자인**
   - 모바일, 태블릿, 데스크톱 최적화
   - 햄버거 메뉴 (모바일)
   - 유동적 그리드 레이아웃

## 🔗 API 엔드포인트

### 견적 문의 API

- **GET** `/tables/quotes?page=1&limit=10&sort=-created_date`
  - 견적 문의 목록 조회
  
- **POST** `/tables/quotes`
  - 새로운 견적 문의 등록
  - Body: JSON (company, name, phone, email, service, message, status)

- **GET** `/tables/quotes/{id}`
  - 특정 견적 문의 상세 조회

- **PATCH** `/tables/quotes/{id}`
  - 견적 문의 상태 업데이트

### 공지사항 API

- **GET** `/tables/notices?limit=10&sort=-created_at`
  - 최신 공지사항 목록 조회

- **POST** `/tables/notices`
  - 새로운 공지사항 등록 (관리자)

## 📝 사용 방법

### 견적 요청하기

1. 메인 페이지에서 "견적요청" 메뉴 클릭
2. 견적 요청 폼 작성
   - 회사명, 담당자명, 연락처, 이메일 입력
   - 서비스 종류 선택
   - 상세 내용 작성
3. 개인정보 수집 동의 체크
4. "견적 요청하기" 버튼 클릭
5. 접수 완료 알림 확인

### 공지사항 확인하기

1. "공지사항" 메뉴 클릭
2. 최신 공지사항 목록 확인
3. NEW 뱃지가 있는 항목은 7일 이내 등록된 공지사항
